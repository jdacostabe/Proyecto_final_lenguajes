Entregables:
 - Gramática: Se encuentra en la carpeta "grammar" y se llama "Python3.g4".
 - Código fuente: Se encuentra en la carpeta src.
 - Ejemplos de uso: Los archivos tanto de entrada como de salida se encuentran en la carpeta "ejemplos".

Para poder ejecutar el programa, se debe tener lo siguiente instalado

Requisitos:
 - InteliJ
 - Java SE Development Kit 13

Con esto, los pasos a seguir para ejecuta el programa es el siguiente:
 1. Abrir la carpeta del proyecto con inteliJ idea.
 2. Configurar la ejecución del main para que se pase como argumento el path y el nombre del archivo a leer, este valor de ejemplo es "input\input.py".
 3. Ejecutar el main del proyecto.
 4. Para ver el archivo generado ingrese a la carpeta input del proyecto y vea el archivo "input.py", el refactor habrá modificado el código ingresado en el mismo archivo.
